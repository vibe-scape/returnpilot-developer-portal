### Overview

This endpoint can be used to retrieve orders.

Endpoint: `/api/v1/orders`

Method: `GET`

<Note>
    Orders are shown in descending order.

    For example, if you had 5 pages of orders, page 1 would contain the latest orders, while page 5 would contain the oldest orders.
</Note>

<PaginationNotice />

### Response

If the orders are successfully retrieved, the API will return a `200 OK` status code.

The response will contain the following content:

<ResponseField name="data" type="array" required>
    An `array` of [Order](/guide/schemas/order) objects.

    If no orders are found, this will be an empty `array`.
</ResponseField>
